Tier 3: Manual Collection

The hardest data to systematize. Requires human curation or specialized extraction.

Source Challenge Approach

Data Freshness Indicators

Every page displays data freshness badges showing when data was last updated:

Current — Updated within 90 days Aging — 90 days to 1 year old Stale — Over 1 year old Sample — Development data, not from live sources

The platform's biggest timeliness constraint is the Census Bureau's 18-24 month lag on comprehensive state finance data. We use FRED quarterly mirrors and NASBO semi-annual surveys to provide more current context where available.
